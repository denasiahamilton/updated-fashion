Color Match

  • 1. Use the color picker to find the closest color hue to your piece of clothing.    

  • 2. Copy the Hex id (six character code) next to the # in the color picker.    

  • 3. Paste the Hex id into the Color Api input box. "#0074AB,rgb(2,45,5)" already in it.)    

  • 4. Press "Scheme For This Color" with a blue background.    

  • 5. Look through the list of analogic, complement, triad, and quad colors.
